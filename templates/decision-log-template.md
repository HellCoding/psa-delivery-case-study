# 의사결정 로그 템플릿

의사결정 로그는 “왜 이 선택을 했는지”를 남기기 위한 문서입니다. 나중에 scope 변경, 일정 변경, baseline 선정 이유를 추적할 수 있도록 근거를 함께 기록합니다.

| 항목 | 값 |
|---|---|
| 결정 ID | DEC-000 |
| 일자 | YYYY-MM-DD |
| 담당자 | |
| 상태 | 제안 / 승인 / 반려 / 대체 |

## 배경

현재 상황, 해결해야 할 문제, 결정이 필요한 이유를 작성합니다.

| | | | |

## 결정 내용

선택한 안과 적용 범위를 명확히 작성합니다.
